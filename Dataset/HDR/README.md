# Description
292 images of SIDD can be downloaded at this [https URL](https://cloud.189.cn/t/yymaUrmuiMny) (size = 398.16MB).

The zip file contains two folders, the 'lowlight' folder and the 'Refimg' folder.

'lowlight' folder contains 292 images under low light.
'Refimg' folder contains 292 images under normal light.

Note: Please do not change the serial number of the images. They correspond to each other. e.g.  'HDRR(1).PNG' is the reference image under normal image of 'HDRL(1).PNG'.
